## Running the file

* start the server as shown below by opening the xamp control pannel and start the Apache
  <img width="986" height="638" alt="image" src="https://github.com/user-attachments/assets/399222a9-4e54-4bbd-9250-2aed8ed2e400" />

* clone the repo :
  ```git clone https://github.com/MohmmedFurqaan/php_workspace.git```

* go to the php_workspace
  ```cd php_workspace```

* Move the folder to your the below specified address:
  ```c:xamp/htdDocs/```

* Now open you browser and type the localhost there type :
  ```http://localhost/php_workspace/```

* edit the file to get the desired output
